# Text-Based Adventure Game (Java)

## Overview
A text-based adventure game developed in Java using object-oriented programming principles. The game allows players to explore rooms, collect items, solve puzzles, and track progress through inventory, map, and scoring systems.

## Features
- Room-based exploration system
- Command-line player interaction
- Inventory management
- Puzzle and progression mechanics
- Score tracking
- Text-based map display
- Multiple interacting Java classes using OOP

## Technologies Used
- Java
- Object-Oriented Programming
- Command-Line Interaction

## Project Structure
- `Game.java` — main game logic and command handling
- `Room.java` — room data and descriptions
- `Position.java` — room coordinates
- `Map.java` — text-based map system
- `Inventory.java` — item storage and management
- `Score.java` — score calculation and tracking

## Purpose
This project was developed to strengthen understanding of object-oriented programming, class design, command handling, and game state management in Java.
